# React E-Commerce Store
- A responsive e-commerce web application built with React.

## Features
- Product Listing
- Categories
- Product Details
- Shopping Cart
- Favorites
- Search Functionality
- Local Storage
- Responsive Design

## Technologies Used
- React
- React Router
- Context API
- CSS3
- DummyJSON API
- React Hot Toast
- React Icons

## Installation
```bash
npm install
npm run dev
```
